Kairos Chart Wheel

    Compute in your browser

    Runs the Kairos engine locally via WebAssembly — no server, nothing leaves this page.

    Location

    Get JSON from the engine: npx -y kairos-astrology compute '<request>' — or click Load Example, or use the compute form above. See docs/example.md for a worked walkthrough.
